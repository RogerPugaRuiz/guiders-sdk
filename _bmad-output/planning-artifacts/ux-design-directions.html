Comparación detallada

Analiza cada dirección según los criterios de decisión más relevantes para el cliente SDK.

A Concierge Clásico

HeaderColor sólido primary (#2563eb)
DensidadMedia — padding 14px
BurbujasRadius 16px + border visible
Nombre IA"✦ Asistente IA" (texto completo)
HandoffSeparador mensaje sistema gris
Quick repliesChips outline violeta
DisclaimerFooter slate-50, visible
PersonalizaciónHeader cambia con --guiders-primary
Tono percibidoProfesional, corporativo
Ideal para
SaaS B2B, servicios financieros, salud, educación. Clientes que priorizan confianza sobre estética.

B Concierge Minimal

HeaderBlanco con border sutil
DensidadBaja — padding 20px, gaps amplios
BurbujasRadius 18px, sin border, colores soft
Nombre IA"✦ IA" (minimal)
HandoffSeparador con líneas finas, texto narrativo
Quick repliesChips borde neutro, hover violeta
DisclaimerFooter muy sutil, casi invisible
PersonalizaciónAvatar usa --guiders-primary, header neutro
Tono percibidoPremium, moderno, calmo (Linear-style)
Ideal para
E-commerce premium, lifestyle, tecnología de consumo, startups. Clientes que priorizan marca y experiencia visual.

C Concierge Express

HeaderCompacto, sin color (10px padding)
DensidadAlta — padding 8px, gap 2px
BurbujasRadius 14px, mini-avatars inline
Nombre IAMini-avatar "IA" (2 letras)
HandoffTexto inline sin separador visual
Quick repliesChips pequeños, border neutro
DisclaimerFooter muy corto "✦ IA · puede contener errores"
PersonalizaciónAvatar usa --guiders-primary
Tono percibidoFuncional, eficiente, técnico
Ideal para
E-commerce de alto volumen, soporte técnico, plataformas con muchos mensajes. Clientes que priorizan velocidad.
