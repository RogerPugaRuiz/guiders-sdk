🎨 Mensajes de Bienvenida con CDN

Ejemplos de configuración para diferentes escenarios

📋 Esta página demuestra:

  • ✅ Mensaje de bienvenida por defecto (sin configuración)
  • ✅ Personalización vía GUIDERS_CONFIG
  • ✅ Cambios dinámicos en tiempo real
  • ✅ Diferentes estilos y idiomas

🚀 Ejemplo 1: Configuración Mínima (Mensaje por Defecto)

Con solo incluir el script, obtienes un mensaje de bienvenida amigable automáticamente:

<script src="https://cdn.guiders.es/sdk/latest/index.js?apiKey=tu-api-key"></script>

Resultado: Mensaje amigable en español con emojis y consejos.

⚙️ Ejemplo 2: Configuración Personalizada

Personaliza el mensaje antes de cargar el SDK:

<script> window.GUIDERS_CONFIG = { apiKey: 'tu-api-key', welcomeMessage: { enabled: true, style: 'professional', language: 'en', includeEmojis: false, showTips: true } }; </script> <script src="https://cdn.guiders.es/sdk/latest/index.js"></script>

💼 Ejemplo 3: Mensaje Personalizado para E-commerce

<script> window.GUIDERS_CONFIG = { apiKey: 'tu-api-key', welcomeMessage: { enabled: true, style: 'custom', customMessage: '¡Hola! 🛍️ Bienvenido a nuestra tienda. ¿Te ayudo a encontrar algo específico?', language: 'es', includeEmojis: true, showTips: false } }; </script> <script src="https://cdn.guiders.es/sdk/latest/index.js"></script>

🏥 Ejemplo 4: Mensaje para Servicios de Salud

<script> window.GUIDERS_CONFIG = { apiKey: 'tu-api-key', welcomeMessage: { enabled: true, style: 'custom', customMessage: '¡Hola! 🏥 Bienvenido a nuestro centro de atención. Estoy aquí para ayudarte con citas, información sobre servicios o consultas generales. ¿Cómo puedo ayudarte hoy?', language: 'es', includeEmojis: true, showTips: true } }; </script> <script src="https://cdn.guiders.es/sdk/latest/index.js"></script>

⚠️ Importante:

Asegúrate de definir window.GUIDERS_CONFIG antes de cargar el script del SDK. Una vez inicializado, puedes usar window.guiders.updateWelcomeMessage() para cambios dinámicos.
